# This project aims to compare the Gillespie algorithm approach to Acid-Base indicator equilibrium against traditional deterministic ODE approach 
Attached files include: python script, data sets for acid, alkali, and buffer solutions named acidspec1, alkalispec1, and bufferspec1.
A write-up is there for optional reading *please read, I think it's quite a good write-up by me*.
There is a separate figures file for ALL figures created by the code.
